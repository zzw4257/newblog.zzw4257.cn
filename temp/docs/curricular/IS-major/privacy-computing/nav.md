# 隐私计算课程导航

## 课程概览

隐私计算 (Privacy Computing) 是一门研究如何在保护数据隐私的同时进行有效计算的学科，结合了密码学、分布式计算和安全多方计算等领域的技术。本导航页面提供对所有章节内容的快速访问。
