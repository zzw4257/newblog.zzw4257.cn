# **📌 一、市场调研流程（总体流程图梳理）**

  

流程从“界定问题”到“跟踪”，环环相扣，可分为 **前期准备 → 调研执行 → 数据处理与报告 → 后续跟踪** 四大阶段。

  

## **（1）前期准备阶段**

1. **界定问题（文献研究）**
    
    明确企业/项目需要解决的核心问题，进行理论与背景资料搜集。
    
2. **确定调研目标（访谈企业）**
    
    进一步与企业沟通，明确此次调研需要达到的目标。
    
3. **设计调研内容（访谈用户）**
    
    确定需要了解的主题、变量、信息类型。
    
4. **设计调研方法（观察、访谈用户）**
    
    → 定性？定量？问卷？深访？小组座谈？观察？
    
    方法随内容与资源不同而选。
    
5. **设计调研样本（撰写调研计划书）**
    
    包括样本抽样方法、样本量、样本结构等。
    
6. **经费预算（签订合同、设计问卷）**
    
    根据调研规模做时间与成本估算，进一步落实问卷设计。
    
7. **人员培训**
    
    对调查员进行访谈技巧、问卷说明、操作流程的培训。
    

---

## **（2）调研执行阶段**

8. **实施调查（问卷调查/访谈/观察等）**
    
    调查员现场或线上执行数据收集。
    

---

## **（3）数据处理与分析阶段**

9. **数据处理**
    
    包含录入、清洗、编码、逻辑检验等。
    
10. **统计分析**
    
    根据调研目标进行描述性分析、推断分析等。
    
11. **撰写报告（第二次文献研究）**
    
    将分析结果结合文献理论，撰写完整研究报告。
    

---

## **（4）后续阶段**

12. **跟踪**
    
    跟进调研报告实施后的效果，必要时再次调研或优化方案。

# **📌 二、影响调研方法选择的因素**

  

（图片中的“四要素”）

1. **调研内容**
    
    内容偏复杂 → 多用定性；内容偏量化 → 多用问卷。
    
2. **经费预算**
    
    预算少 → 使用线上问卷、电话调查；
    
    预算多 → 可采取深度访谈、实地观察、抽样调查等。
    
3. **时间要求**
    
    时间紧 → 快速抽样或线上调查
    
    时间宽裕 → 可使用复杂设计或多方法组合。
    
4. **数据质量要求**
    
    要求高 → 采用严谨抽样、标准化流程
    
    要求基础 → 可用便利样本、快速访谈。

# **📌 四、企业选择市场调研公司的原则**

  

企业在选择时应重点考虑：

- 🔸 **调查经验如何？**（行业与方法经验）
    
- 🔸 **调研服务的声誉如何？**
    
- 🔸 **能否按时完成调研项目？**
    
- 🔸 **能否与客户良好沟通并处理矛盾？**
    
- 🔸 ……

# **📌 五、市场调研行业结构（图示四层）**

  

行业角色由上到下分为四类：

  

### **1. 信息用户**

  

企业、政府、机构 —— 使用调研结果进行决策。

  

### **2. 调研提供者**

  

专业调研公司，负责设计研究、分析数据、输出报告。

  

### **3. 数据收集者**

  

调查员、第三方数据采集公司等。

  

### **4. 被访者**

  

问卷填写者、受访用户、目标样本群体。

---

# **✅ 总结（简化记忆版）**

- **调研流程**：界定问题 → 设目标 → 设计内容/方法/样本 → 预算 → 培训 → 调查 → 数据处理 → 分析 → 报告 → 跟踪
    
- **方法选择四要素**：内容、预算、时间、数据质量
    
- **调研公司选择**：经验、声誉、准时性、沟通能力
    
- **行业结构**：信息用户 → 调研提供者 → 数据收集者 → 被访者
